Frequently Asked Questions

Will memory data be uploaded to servers?

No. All memory data is stored in your browser's localStorage, and our servers cannot access this data. Only when generating responses, relevant memories will be sent to the AI model as context, but will not be permanently saved.

How to ensure memory accuracy?

AI will analyze the importance of conversation content through the DeepSeek model and only record explicit personal information and preferences. If you find incorrect memories, you can always view and delete them in settings.

What happens to existing memories when I turn off the memory feature?

Turning off the memory feature only stops new memories from being generated; existing memories will remain. If you want to completely clear them, you need to manually click the "Clear All Memories" button.

How much storage space does memory data occupy?

The system stores a maximum of 50 memories, each usually not exceeding 1KB, totaling no more than 50KB. When storage space is insufficient, the system will automatically clear the oldest memories.

Will memories remain when switching devices or browsers?

Memory data is tied to your browser. You'll need to rebuild memories when switching devices or browsers. It's recommended to regularly export important memory data for backup.
