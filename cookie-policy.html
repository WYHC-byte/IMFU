About Cookies

Last Updated: August 6, 2024

Cookies are small text files stored in your browser that help improve your experience. This policy explains how we use cookies and similar technologies.

Types of Cookies We Use

Necessary Always Enabled

These cookies are essential for the website to function properly and cannot be disabled in our systems.

Cookie Name Purpose Duration
darkMode Stores interface theme preference Persistent
tyloUserId Unique user identifier Persistent
tyloLanguage Language settings Persistent
Functional

These cookies enable enhanced functionality and personalization.

Cookie Name Purpose Duration
tyloChatHistory Saves chat history Persistent
tyloMemorySystem AI memory functionality Persistent
tyloUserNickname User nickname Persistent
tyloAchievements Achievement system data Persistent
Analytics

These cookies help us understand how visitors use our website to improve our services.

Currently, we don't use third-party analytics cookies. If we introduce them in the future, we will update this policy and seek your consent.

Local Storage

In addition to cookies, we use browser LocalStorage to store data. LocalStorage is similar to cookies but can store more data and isn't sent to the server with each request.

Stored Data

  • • Chat history
  • • AI memory content
  • • User settings and preferences
  • • Points and level information

Data Characteristics

  • • Stored locally only
  • • Not sent to the server
  • • Can be cleared anytime
  • • Capacity about 5-10MB

How to Manage Cookies and Local Storage

Through Our Settings

Click the "Manage Cookie Preferences" button at the top of this page to select which cookie types you want to enable.

Through Browser Settings

Most browsers allow you to control cookie settings:

  • • Chrome: Settings → Privacy and security → Cookies
  • • Firefox: Settings → Privacy & Security → Cookies and Site Data
  • • Safari: Preferences → Privacy → Cookies and Website Data
  • • Edge: Settings → Privacy, search, and services → Cookies

Clear All Data

You can clear all stored data by:

  1. Going to TyloAI settings
  2. Clicking "Delete all conversations"
  3. Clicking "Clear all memories"
  4. Or directly clearing your browser cache
